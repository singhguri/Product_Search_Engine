# BEST BUY PRODUCT SEARCH ENGINE

## Features

- Page Ranking
- Word Completion & Suggestion
- Product Search
- Inverted Indexing
- Search Frequency
- Spell Checker
- Pattern Matching
- Web Crawler
- Web Scraper

## Dependencies

The dependencies for this project must be same as listed below:
- JDK-17
- JSoup-1.15.3

## How to run?

- Install the the dependencies.
- Run ```main.java```
